# ConvertFrom-FileTime

## SYNOPSIS
Converts Win32 FILETIME (long) into date

## SYNTAX

```
ConvertFrom-FileTime [-FileTime] <Int64> [<CommonParameters>]
```

## DESCRIPTION
Converts Win32 FILETIME (long) into date

## EXAMPLES

### EXAMPLE 1
```
ConvertFrom-FileTime 132664585367265239
```
